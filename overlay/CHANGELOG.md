# Overlay Changelog

Este archivo es leído por Nopolo para mostrar al usuario **por qué** se actualizaron
los archivos del overlay cuando entra en conflicto con sus ediciones personales.

## Formato

Cada sección empieza con `## nombre_del_archivo` y puede tener varias líneas de descripción.
El texto se muestra en el popup de conflictos al iniciar la app.

---

## overlay.html

Estructura principal del overlay actualizada.
Revisa los nuevos elementos añadidos antes de restaurar tu versión personalizada.

## overlay.css

Estilos visuales actualizados.
Compara con tu `.old` para combinar tus colores y animaciones con los nuevos estilos base.
